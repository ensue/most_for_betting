# Research — thesis, data, fair probability

This folder is the **lab**: structured work that turns information into **probability estimates** you can compare to the market.

MOST is not only accountability. It is also:

- **Research** — sources, factors, what would change your mind
- **Estimation** — fair probability (or range), vs implied probability from odds
- **Calibration** — outcomes feed Brier score and `journal/bets/_summary.md`

Use `TEMPLATE.md` for new notes. File names: `YYYY-MM-DD-<short-slug>.md`.

The AI should write substantive research here when you analyze an event or build a thesis — not only reflections in `journal/reflections/`.
